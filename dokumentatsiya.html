Telegram Bot uchun Hujjat

Botni sozlash va ishga tushirish bo'yicha bosqichma-bosqich qo'llanma.

1-qadam: Zaruriy kutubxonalarni o'rnatish

Node.js kompyuteringizga o'rnatilganiga ishonch hosil qiling. So'ngra, quyidagi buyruqni bajarib, kerakli kutubxonalarni o'rnating:

npm install telegram fs path

2-qadam: API ID va API Hash olish

Telegram API ID va API Hashni olish uchun quyidagi bosqichlarni bajarish kerak:

  • Telegramning r…ôsmi saytiga kiring.
  • Hisobingiz bilan tizimga kiring.
  • "API Development Tools" bo'limiga o'ting.
  • Yangi ilova yaratish uchun Create new application tugmasini bosing.
  • Ilovangiz nomini va qisqacha tavsifini kiriting, so'ngra Create application tugmasini bosing.
  • Ilova yaratildi va sizga API ID va API Hash beriladi. Ushbu ma'lumotlarni o'zingizning index.js faylingizda apiId va apiHash sifatida qo'llang.

3-qadam: Skriptni sozlash

index.js faylini oching va quyidagi qatorlarga o'zingizning ma'lumotlaringizni kiriting:

const apiId = YOUR_API_ID; // O'zingizning API ID
const apiHash = "YOUR_API_HASH"; // O'zingizning API Hash
const channelOrGroupID = "-1001234567890"; // Guruh yoki kanal ID

4-qadam: Skriptni ishga tushirish

Quyidagi buyruq orqali botni ishga tushiring:

node index.js

5-qadam: Jarayonni kuzatish

Bot ishga tushirilgandan so'ng, konsolda foydalanuvchilar chiqarilayotgani va ularning ma'lumotlari User.json fayliga yozilayotgani haqidagi ma'lumotlarni ko'rishingiz mumkin.

Savollar bormi? Telegram Bot Hujjatlarini ko'rib chiqing.
